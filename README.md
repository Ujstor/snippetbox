# SnippetBox


This repository contains code examples and projects from Alex Edwards [book](https://lets-go.alexedwards.net/)


- **Fundamentals**: Start a server, handle requests, and serve static files.
- **Structure**: Organize your code for scalability.
- **Dependencies**: Manage and version control with Go Modules.
- **Configuration**: Handle settings with command-line flags and dependency injection.
- **Error Handling**: Implement leveled logging and centralized error handling.
- **Database**: Design models and execute SQL queries.
- **HTML Templating**: Cache templates and handle dynamic data.
- **Middleware**: Create middleware for common tasks.
- **RESTful Routing**: Design routes following REST principles.
- **Form Validation**: Implement reusable patterns for form validation.
- **Session Management**: Use sessions to persist data.
- **HTTPS**: Set up HTTPS for improved security.
- **File Embedding**: Embed assets into your Go binary.
- **Security**: Prevent common vulnerabilities.
- **Authentication & Authorization**: Encrypt passwords and handle user authentication.
- **Testing**: Create unit, integration, and end-to-end tests.
